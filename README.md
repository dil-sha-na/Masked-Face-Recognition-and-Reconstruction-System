# Masked Face Recognition and Reconstruction System

## Overview
A real-time masked face recognition and facial reconstruction system developed using Python and OpenCV.

## Features
- Real-time masked face detection
- Face recognition using ORB and KNN
- IoU-based tracking
- Facial reconstruction
- Flask-based web interface

## Technologies Used
- Python
- OpenCV
- Flask
- SSD
- ORB
- KNN
- NumPy

## Installation

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```

## Applications
- Surveillance Systems
- Security Monitoring
- Identity Verification
